# รายงานการวิเคราะห์ข้อมูลสุขภาพการนอนและการดำเนินชีวิต (Sleep Health and Lifestyle Analysis)

โปรเจกต์นี้เป็นการวิเคราะห์เชิงปริมาณเพื่อศึกษาปัจจัยต่างๆ ที่ส่งผลต่อคุณภาพการนอนหลับ โดยใช้ชุดข้อมูล **Sleep Health and Lifestyle Dataset** ผ่านการวิเคราะห์ด้วยภาษา R เพื่อหาความสัมพันธ์และเปรียบเทียบปัจจัยด้านไลฟ์สไตล์ อาชีพ และสภาพร่างกาย

## วัตถุประสงค์ (Objectives)
1. เพื่ออธิบายลักษณะทั่วไปของข้อมูลสุขภาพการนอนและการดำเนินชีวิตด้วยสถิติเชิงพรรณนา
2. เพื่อศึกษาความสัมพันธ์ระหว่างปัจจัยต่างๆ (เช่น ความเครียด, กิจกรรมทางกาย) กับคุณภาพการนอน
3. เพื่อเปรียบเทียบความแตกต่างของคุณภาพการนอนระหว่างกลุ่มเพศ, กลุ่ม BMI และกลุ่มอาชีพ
4. เพื่อระบุปัจจัยที่มีอิทธิพลต่อคุณภาพการนอนอย่างมีนัยสำคัญทางสถิติ

## ข้อมูลโดยสรุป (Data Summary)
* **กลุ่มตัวอย่าง:** จำนวน 186 รายการ (เพศชาย 70.4%, เพศหญิง 29.6%) ช่วงอายุ 29-59 ปี
* **ระยะเวลาการนอนเฉลี่ย:** 7.08 ชั่วโมง
* **คุณภาพการนอนเฉลี่ย:** 7.2 คะแนน (จากเต็ม 9)
* **ดัชนีมวลกาย (BMI):** ส่วนใหญ่เป็นกลุ่ม Normal (86.56%)

## ข้อค้นพบหลัก (Key Findings)

### 1. ปัจจัยที่มีผลต่อคุณภาพการนอน (Correlation Analysis)
จากการทดสอบด้วย **Pearson Correlation** พบว่า:
* **ระดับความเครียด (Stress Level):** มีความสัมพันธ์เชิงลบสูงมากกับคุณภาพการนอน (r = -0.87) หมายความว่ายิ่งเครียดมาก คุณภาพการนอนจะลดลงอย่างเห็นได้ชัด
* **ระยะเวลาการนอน (Sleep Duration):** มีความสัมพันธ์เชิงบวกสูง (r = 0.73) ยิ่งนอนนานขึ้น คุณภาพการนอนจะดีขึ้น
* **กิจกรรมทางกาย (Physical Activity):** มีความสัมพันธ์เชิงบวก (r = 0.67) การออกกำลังกายส่งผลดีต่อคุณภาพการนอน

### 2. การเปรียบเทียบกลุ่มตัวอย่าง (Comparison Analysis)
* **เพศ (Gender):** ผลการทดสอบ T-test พบว่าเพศชายและเพศหญิง **ไม่มีความแตกต่าง** ของคุณภาพการนอนอย่างมีนัยสำคัญทางสถิติ (p > 0.05)
* **กลุ่ม BMI:** กลุ่ม **Overweight** มีคุณภาพการนอนต่ำกว่ากลุ่ม Normal อย่างมีนัยสำคัญ (p < 0.05) และมีความเสี่ยงต่อภาวะหยุดหายใจขณะหลับ (Sleep Apnea) มากกว่า
* **อาชีพ (Occupation):** อาชีพที่มีภาระงานสูง เช่น **นักวิทยาศาสตร์, พยาบาล และแพทย์** มีคุณภาพการนอนต่ำกว่าอาชีพอื่นๆ อย่างชัดเจน
